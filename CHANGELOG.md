# 更新日志

## v0.10.1 — 多会话历史聊天

- **按会话隔离**: 同一本书可创建多个独立聊天会话，每个会话历史独立存储。
- **会话标签栏**: 聊天面板顶部标签栏切换会话，可新建/删除/重命名。
- **默认行为**: 打开书自动创建新会话，不混入旧对话。

## v0.10 — 书架自由布局

- **真实书架绝对定位**: 每个学科书架可自由拖动到任意位置，不吸附固定网格。
- **碰撞弹开**: 拖拽释放时若与其他书架重叠，自动计算最近可用空位并弹开。
- **布局持久化**: 书架位置实时保存到本地，重启后恢复。
- **书换分类**: 直接把书脊拖到另一个书架上，松开即自动更换学科分组。

## v0.9 — 统计面板与小组件

- **研读周报**: 环形进度条（目标 20h）、累计阅读时长、比上周对比、翻译批注数、AI 生成摘要数。数据按自然周聚合，本地持久化。
- **阅读热力图**: 8 周×7 天方块网格，颜色深浅映射每日阅读时长。
- **连续阅读**: 火焰图标 + 大号天数，自动计算连续阅读天数与历史最高纪录。
- **导入进度面板**: 顶部常驻「正在处理」书籍列表，每本带状态 emoji + 渐变进度条 + 百分比。
- **自由小组件**: 配置面板可勾选添加/删除组件；右下角原生 resize handle 无极调节大小，flex wrap 自由布局，不吸附网格。

## v0.8.2 — 阅读器增强

- **截图问 AI 四阶段动画**: Siri 式彩虹光带环绕 → 页面灵魂化 → 飞行 → 爆开消散。
- **PDF 原地缩放终版**: 触控板双指捏合平滑缩放（20%–500%），50ms 节流 + 25% 惯性保留；`flushSync` + 锚点同步恢复滚动位置，缩放不跳页。
- **双页模式截图**: 书本模式下合并左右两页 canvas，截图覆盖整面。
- **LaTeX 公式渲染修复**: 自动将模型输出的 `\(...\)` / `\[...\]` 转换为 `$...$` / `$$...$$`，兼容 KaTeX。

## v0.8.1 — 页码修复

- 修复知识地图页码错位：强制模型只认注入的 `<<PDF_PAGE=N>>` 标记，并在解析时用 `snapToValidPage` 把返回页码吸附到真实物理页。

## v0.8.0 — 书架与整理

- 主界面「真实书架」：每个学科一格搁板，书以彩色书脊竖立其上，三种装帧样式，竖排书名，悬停抽起。
- 「正在处理」进度面板：每本带大号进度条 + 阶段文字 + 百分比，可取消/移除。
- 导入时本地即时清理文件名。
- 配置 API 后，导入会用 AI 优化书名并判定学科；支持「AI 整理书架」批量整理。

## v0.7 — 学术级回答 + Markdown 渲染

> ⚠️ 本版新增了依赖，请先在项目目录重新执行 `npm install`，再 `npm run dev`。

- **学术级双语解答**: 重写提问/学习两套系统提示词。要求准确、深入、系统、完整——讲定义、前提、原理/推导、公式与符号含义、适用范围、易错点与知识联系。学习模式按「概览/核心概念/原理推导/公式符号/例子应用/易错点/术语表」结构系统讲解。
- **检索召回量提高**: 学习模式 12 段、提问模式 8 段。
- **完整 Markdown 渲染**: 接入 react-markdown + remark-gfm，支持标题、列表、表格、引用、代码块、链接等全部 Markdown 语法。
- **数学/物理公式**: 接入 remark-math + KaTeX，行内 `$...$`、独立 `$$...$$` 公式都能正确排版。
- **术语高亮**: 「中文(English)」形式在 Markdown 渲染下依旧保留。

新增依赖：`react-markdown`、`remark-gfm`、`remark-math`、`rehype-katex`、`katex`。

## v0.6 — 详细知识地图

- 大纲升级为三级结构 — **章 / 节 / 知识点**，每个知识点附一句话中文要点。
- 通读全书再提炼：按页范围分批（每批约 18 页）map-reduce，模型实际读完全书内容。
- 侧栏「知识地图」支持三级缩进、知识点要点展示、章节折叠/展开、一键重新生成。

## v0.5 — 去掉 Embedding 依赖

- 检索改为**本地 BM25 词法检索**（零额外 API、离线、秒建），只要服务能对话就能用。
- **跨语言检索**: 中文提问时，先用对话模型把问题转成英文关键词，再与原问题一起在 BM25 上召回。

## v0.4 — PDF 阅读增强

- PDF 缩放改为无极调节（20%–500%）。
- 新增「适宽 / 整页」自适应，不规则尺寸也能完整显示。
- 连续翻页更顺、不闪白：用各页真实尺寸做精确占位，预渲染范围加大到上下约 1.5 屏。
- 模型调用 404/HTML 报错给出可操作提示。

## v0.3 — OCR 与渲染修复

- 放宽 CSP，放行 Tesseract 的 blob worker；显式指定 Tesseract 路径。
- 本地 OCR 内核起不来时，hybrid 模式自动整本改走云端。
- 大 PDF 导入改为逐页让出事件循环，实时上报进度。
- PDF 渲染改为按设备像素比（Retina）绘制。
- 新增连续滚动视图与书本对页视图。
